# 🚀 Smart Resume Advisor

A full-stack Java application that analyzes resumes against job descriptions and provides skill-based match scores with personalized improvement suggestions.

---

## ✨ Features

- 📄 Resume Upload (PDF)
- 🎯 ATS-style Resume Matching
- 📊 Skill Match Score
- 🧠 Missing Skills Detection
- 📈 Resume Analysis Dashboard
- 👤 User Authentication
- 💾 Database Integration

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/Renuka-Ratnala/smart-resume-advisor.git
```

### Backend

```bash
mvn spring-boot:run
```

### Frontend

```bash
npm install
npm run dev
```
